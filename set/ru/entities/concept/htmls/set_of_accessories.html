Множество принадлежности М - это линейно упорядоченное множество, элементы которого являются значениями функции принадлежности нечеткого множества A. В качестве M выбирается отрезок [0, 1]. Если M={0,1} (то есть состоит только из двух элементов), то нечёткое множество может рассматриваться как обычное, чёткое множество.
