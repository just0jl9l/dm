Формальное определение операции пересечение множеств
A ∩ B = { x | x ∈ A ∧ x ∈ B}

Дополнительные примеры

  • {1, 2, 3} ∩ {3, 4, 5} = {3}
  • {а, б} ∩ {3, 4} = ∅
  • {1, 2} ∩ ∅ = ∅

    Свойства операции пересечение
  • А ∩ ∅ = ∅ - Закон Идентичности
  • А ∩ U = A - Доминирование закон
  • A ∩ A = А - Идемпотентный закон
  • А ∩ B = B ∩ А - Коммутативный закон
  • А ∩ (B ∩ C) = (А ∩ B) ∩ C - Ассоциативный закон.
