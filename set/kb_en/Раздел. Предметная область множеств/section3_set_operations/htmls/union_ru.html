Формальное определение операции объединение множеств
A U B = { x | x ∈ A ∨ x ∈ B}

Дополнительные примеры

  • {1, 2, 3} ∪ {3, 4, 5} = {1, 2, 3, 4, 5}
  • {а, б} ∪ {3, 4} = {а, б, 3, 4}
  • {1, 2} ∪ ∅ = {1, 2}

    Свойства операции объединение множеств
  • А ∪ ∅ = А - Закон Идентичности
  • А ∪ U = A - Доминирование закон
  • A ∪ A = А - Идемпотентный закон
  • А ∪ B = B ∪ А - Коммутативный закон
  • А ∪ (B ∪ C) = (А ∪ B) ∪ C - Ассоциативный закон.
