Формальное определение дополнения множества:
A = { x | x ∉ A } = Ac ∨ U – A,
где U является универсальным множества

Дальнейшие примеры (в предположении, что U = Z)
{1, 2, 3}c = { ... , -2, -1, 0, 4, 5, 6, ... }
{a, b}c = Z

Свойства данного множества

  • (Ac)c = A - Закон отрицания
  • A ∪ Ac = U - Закон дополнения
  • A ∩ Ac = ∅ - Закон дополнения
