Formal definition for the intersection of two sets
A ∩ B = { x | x ∈ A ∧ x ∈ B}
A ∩ B = { x | x ∈ A ∧ x ∈ B}

Further examples

  • {1, 2, 3} ∩ {3, 4, 5} = {3}
  • {а, б} ∩ {3, 4} = ∅
  • {1, 2} ∩ ∅ = ∅

    Properties of the intersection operation
  • А ∩ ∅ = А - Identity law
  • А ∩ U = A - Domination law
  • A ∩ A = А - Idempotent law
  • А ∩ B = B ∩ А - Commutative law
  • А ∩ (B ∩ C) = (А ∩ B) ∩ C - Associative law.
