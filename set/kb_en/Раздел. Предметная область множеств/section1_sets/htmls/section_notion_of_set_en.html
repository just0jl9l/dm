A set is a unordered collection of “objects”: -eople in a class: {Alice, Bob, Chris } -States in the US: {Alabama, Alaska, Virginia, … } -Sets can contain non-related elements: {3, a, red, Virginia } We will most often use sets of numbers: -All positive numbers less than or equal to 5: {1, 2, 3, 4, 5} -A few selected real numbers: { 2.1, π, 0, -6.32, e }
br Order does not matter
{1, 2, 3, 4, 5} is equivalent to {3, 5, 2, 4, 1}

  • It does not make sense to list somebody twice
