Если множество не имеет элементов, то оно называется пустым множеством
Для его обозначения используется символ Ø
Таким образом, Ø = {} - Запомнить!
пустое множество может быть элементом другого множества:
{Ø , 1, 2, 3, х}

Ø ≠ {Ø}
Первое множество нулевых элементов
Второе множество из 1 элемента
Заменить, если взяь Ø по {}, и вы получите: {} {≠ {}}
