How to proof a set identity

For example: A ∩ B = B - (B - A)
Four methods:

  • Use the basic set identities
  • Use membership tables
  • Prove each set is a subset of each other
  • Use set builder notation and logical equivalences

Proofs examples

(A U B) ⊆ (A U B U C)
If x ∊ (A U B). Then x ∊ A or x ∊ B, or both.
Then x ∊ (A U B U C), because (A U B U C) contains both A and B. Hence (A U B) ⊆ (A U B U C).

(A ∩ B ∩ C) ⊆ (A ∩ B)
x ∊ (A ∩ B ∩ C). Then x ∊ A, x ∊ B and x ∊ C.
x ∊ A and x ∊ B, x ∊ (A ∩ B). x is chosen arbitrarily. Hence, (A ∩ B ∩ C) ⊆ (A ∩ B)

(A-B)-C ⊆ A-C
(A ∩ Bc) ∩ Cc ⊆ A ∩ Cc
Solution is similar to the one above
(A-C) ∩ (C-B) = ∅
A ∩ Cc ∩ C ∩ Bc
A ∩ ∅ ∩ Bc
= ∅

= A ∩ (A ∩ Bc)c
= A ∩ (Ac ∪ B)
= (A ∩ Ac) ∪ (A ∩ B)
= ∅ ∪ (A ∩ B)
= A ∩ B

Showing each is a subset of the others
(A ∩ B)c = ∪ Bc
Proof: Want to prove that
(A ∩ B)c ⊆ Ac ∪ Bc and (A ∩ B)c ⊇ Ac ∪ Bc
x ∈ (A ∩ B)c
⇒ ¬ x ∈ (A ∩ B)
⇒ ¬ (x ∈ A ∩ B)
⇒ ¬ (x ∈ A ∧ x ∈ B)
⇒ (x ∉ A) ∨ (x ∉ B)
⇒ x ∈ ∨ x ∈ Bc
⇒ x ∈ ∪ Bc
