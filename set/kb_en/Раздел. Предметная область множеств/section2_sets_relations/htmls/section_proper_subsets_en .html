If each element of set A are also elements of a set B, but set B contain at least one element, not included in A, that is A ⊆ B, and A≠B, then A is proper subset B. A proper subset is written as A ⊂ B, B ⊃ A. The difference between “subset” and “proper subset” is like the difference between “less than or equal to” and “less than” for numbers. Example:

  • Let T = {0, 1, 2, 3, 4, 5}
  • If S = {1, 2, 3}, S is not equal to T, and S is a subset of T
  • Then S ⊂ T
  • Let Q = {4, 5, 6}. Q is neither a subset of T nor a proper subset of T
