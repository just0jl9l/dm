If each element of set A are also elements of a set B, then A is a subset of B. This is specified by A ⊆ B, B ⊇ A, meaning that ∀ Ñ… (x ∈ A -> x ∈ B). Obviously, if A ⊆ B, and B ⊆ A, then A = B. The empty set is a subset of all sets. Examples:

  • {1, 2, 3, 4, 5} ⊇ {1, 2, 3, 4}
  • {2, 4, 6} ⊆ {1, 2, 3, 4, 5, 6, 7}
For any set S
  • ∀ S (S ⊆ S)
  • ∀ S (∅ ⊆ S)
