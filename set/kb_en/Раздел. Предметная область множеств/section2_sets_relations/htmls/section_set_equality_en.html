If sets A and В have the same elements and cardinalitis, then this sets are equal А=В. Examples:

  • {1, 2, 3, 4, 5} = {5, 4, 3, 2, 1}
  • {1, 2, 3, 2, 4, 3, 2, 1} = {4, 3, 2, 1}
Two sets are not equal if they do not have the same elements, written as А≠В. Example:
  • {1, 2, 3, 4, 5} ≠ {1, 2, 3, 4}
