Если множества A и В содержат одни и те же элементы и мощности их равны, то эти множества считаются равными. Это обозначается так: А=В. Пример:

  • {1, 2, 3, 4, 5} = {5, 4, 3, 2, 1}
  • {1, 2, 3, 2, 4, 3, 2, 1} = {4, 3, 2, 1}
Неравные множества состоят из различных элементов, обозначается так: А≠В. Пример:
  • {1, 2, 3, 4, 5} ≠ {1, 2, 3, 4}
